Cybersecurity Intrusion Detection

This project addresses the critical challenge of detecting cyber threats in network traffic using machine learning. The dataset, sourced from Kaggle, included features like network packet size, login attempts, session duration, IP reputation score, failed logins, unusual time access, protocol type, encryption used, and browser type. The target variable was binary: attack detected (Yes/No).

Model Selection: Random Forest classifier was used for its robustness and ability to handle high-dimensional data. Hyperparameter tuning was performed using Grid Search to optimize performance.

Deployment: The model was deployed as an interactive web application using Gradio and hosted on Hugging Face Spaces for real-time predictions.

Associated with: Student at National Computing Center Education - IPMC, Cybersecurity Short Course.

Phishing Email Detection

This project focuses on detecting phishing emails using machine learning. The dataset sourced from Kaggle included email text and labels. The target variable was binary: phishing (0/1). zero(0) means the email is legitimate and one(1) means its a phishing email.

Model Selection: A RandomForestClassifier model was used for its interpretability and effectiveness in binary classification tasks. Feature engineering included TF-IDF vectorization for text data and tokenization for additional context.

Deployment: The model was deployed as an interactive web application using Gradio and hosted on Hugging Face Spaces for real-time predictions.

Associated with: Student at National Computing Center Education - IPMC, Cybersecurity Short Course.

Home Intruder Detection

This project focuses on detecting intruders in home environments using computer vision. The dataset was collected and annotated using Roboflow, containing images of various home intrusion scenarios. The model was trained to detect and classify potential intruders.

Model Selection: Used YOLOv11 (a pretrained object detection model) for its real-time detection capabilities and high accuracy in identifying persons. Training was performed on Google Colab leveraging GPU acceleration.

Deployment: The model was deployed as an interactive web application using Gradio, allowing users to upload images or use real-time camera input for detection. The application is hosted permanently on Hugging Face Spaces for easy access.

Associated with: Student at National Computing Center Education - IPMC, Cybersecurity Short Course.

Email Spam Detector

This project tackles email spam classification using machine learning. The dataset was sourced from the Enron Email Dataset, featuring raw email content, sender/recipient metadata, and subject lines. Key preprocessing included text cleaning, stopword removal, and TF-IDF vectorization.

Model Selection: Naive Bayes classifier was chosen for its efficiency with text data and probabilistic spam detection capabilities. The model achieved 98% accuracy on test data.

Deployment: Deployed as an interactive web app using Gradio, hosted permanently on Hugging Face Spaces for real-time spam predictions (1 = SPAM, 0 = NOT SPAM).

Associated with: Student at National Computing Center Education - IPMC, Cybersecurity Short Course.
